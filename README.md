## Описание

       Проэкт разработан на чистом js(typescript), c использованием вэб компонентов.
       Архитектура FSD .
       Простенький сервер на node js, база данный mongodb- https://github.com/Ruslan-Gubin/direcroty-backend
       Сборщик Vite.
       
Сайт заметок для технологии JS, который может сохранять до 5 фото и код к посту - это онлайн-сервис, который позволяет пользователям создавать записи о своем опыте работы с JS и делиться ими с другими.

На сайте пользователи могут создавать новые заметки, добавляя текстовое содержание, изображения и фрагменты кода на JavaScript. С помощью функционала загрузки файлов, каждая заметка может содержать до 5 фотографий и неограниченное количество кода.

Кроме того, система автоматически сохраняет все изменения, вносимые в заметку, что обеспечивает защиту данных пользователей в случае возникновения проблем с браузером или компьютером. Пользователи также могут редактировать свои заметки, добавлять новые изображения и код, а также переупорядочивать контент по своему усмотрению.

Данный сайт отлично подходит тем, кто хочет сохранять и делиться своим опытом работы с JavaScript и кому нужны инструменты для удобного хранения фото и кода в одном месте.

## Инструкция по запуску:

          Установка зависимостей: npm install.
          Запуск: npm run start
          Собрать dev: npm run build
          Собрать prod: npm run build-prod
          Тесты: npm run test
